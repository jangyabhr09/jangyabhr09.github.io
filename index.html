Differentiability of a Real function

Definition: let f be defined on an interval I = [a,b] ⊆ ℝ. For any x ∈ [a,b] form the quotient

f(t) - f(x) t - x (a < t < b,t≠x)

& define

f′(x) = lim ⁡ t→xf(t) - f(x) t - x

provided that this limit exists. What we have done here is that we associated with the function f another function f′ whose domain is the set of points x at which the limit exists.

  • If the limit exists at x then f is said to be differentiable.
  • f′ is called the derivative of f.

Theorem 1. If a function is differentiable at a point then it is necessarily continuous at that point.

Proof. let f be defined on [a,b] & let also f be differentiable at c. i.e.

lim ⁡ t→cf(t) - f(c) t - c exists, (a < t < b,t≠x)

Now

f(t) - f(c) t - c ⋅ (t - c) → f′(c) ⋅ 0 = 0 as t → c

So

lim ⁡ t→cf(t) - f(c) t - c (t - c) = 0

⇒ lim ⁡ t→cf(t) - f(c) = 0 ⇒ lim ⁡ t→cf(t) = f(c)

i.e. f is continuous at c. □

However the converse is not true .for example f(x) = |x|, it is continuous at 0 but not differentiable there at. It is also possible to construct a nowhere differentiable function which is continuous every where

f(x) = ∑ n=0∞(3 4)nϕ(4nx)

where ϕ(x) = |x|,(-1 ≤ x ≤ 1) and ϕ(x + 2) = ϕ(x)

Left hand derivative & Right hand derivative

Now consider again the quotient and it’s limit

f′(x) = lim ⁡ t→xf(t) - f(x) t - x (a < t < b,t≠x)

As in limits it is possible to consider right-hand & left-hand limits in above which leads to the notion of left-hand derivative & right hand derivatives,

let x ∈ (a,b) ,then

lim ⁡ t→x+f(t) - f(x) t - x (a < t < b,t > x)

is called right hand derivative and is denoted by f′(x+) or Rf′(x). Similarly, the left hand derivative is

Lf′(x) = f′(x-) = lim ⁡ t→x-f(t) - f(x) t - x (a < t < b,t < x)

Observe that f′(x) exists only when Lf′(x) and Rf′(x) exist and are equal.

Examples.

  1. f(x) = xif0 ≤ x < 1 1ifx ≥ 1

    Lf′(1) = lim ⁡ t→1-f(t) - f(1) t - 1 (t <1)
    = lim ⁡ t→1-t - 1 t - 1 (as t < 1 ⇒ f(t) = t)
    = 1
    Rf′(1) = lim ⁡ t→1+f(t) - f(1) t - 1 (t > 1)
    = lim ⁡ t→1+1 - 1 t - 1 (as t > 1 ⇒ f(t) = 1)
    = 0

    Therefore Lf′(1)≠Rf′(1). So f is not differentiable at x≠1.

  2. f(x) = 2if0 ≤ x ≤ 1 xifx > 1

    Lf′(1) = lim ⁡ t→1-f(t) - f(1) t - 1 (t <1)
    = lim ⁡ t→1-2 - 2 t - 1 (as t < 1 ⇒ f(t) = 2)
    =0
    Rf′(1) = lim ⁡ t→1+f(t) - f(1) t - 1 (t > 1)
    = lim ⁡ t→1+t - 2 t - 1 (as t > 1 ⇒ f(t) =t)
    = ∞

    Therefore Lf′(1)≠Rf′(1). So f is not differentiable at x≠1.
